# Sort

## Counting sort (計數排序法)

## Example
```
Original List 1: 
[0, 2, 5, 3, 7, 9, 10, 3, 7, 6]
Basic counting sorted list of counts: 
[1, 0, 1, 2, 0, 1, 1, 2, 0, 1, 1]
Sorting results based on count list: 
[0, 2, 3, 3, 5, 6, 7, 7, 9, 10]
----------------------------------------
Original List 2: 
[101, 109, 107, 103, 108, 102, 103, 110, 107, 103]
Optimize counting sorted list of counts: 
[1, 1, 3, 0, 0, 0, 2, 1, 1, 1]
將優化後的計數排序列表轉換成每個數字在結果數組中首次出現的位置: 
[101, 102, 103, 103, 103, 107, 107, 108, 109, 110]
----------------------------------------
Original List 3: 
[1, 4, 1, 2, 5, 2, 4, 1, 8]
Basic counting sorted list of counts:
[0, 3, 2, 0, 2, 1, 0, 0, 1]
將優化後的計數排序列表轉換成每個數字在結果數組中首次出現的位置: 
[0, 3, 5, 5, 7, 8, 8, 8, 9]
對應原本數組的位址表: 
[0, 5, 1, 3, 7, 4, 6, 2, 8]
利用位址表來產生排序: 
[1, 1, 1, 2, 2, 4, 4, 5, 8]
----------------------------------------
Original List 4: 
[-1, 4, -1, -2, 5, -2, 4, -1, 8]
Optimize counting sorted list of counts:
[-2, -1, 0, 1, 2, 3, 4, 5, 6, 7, 8]
[2, 3, 0, 0, 0, 0, 2, 1, 0, 0, 1]
將優化後的計數排序列表轉換成每個數字在結果數組中首次出現的位置: 
[-2, -1, 0, 1, 2, 3, 4, 5, 6, 7, 8]
[2, 5, 5, 5, 5, 5, 7, 8, 8, 8, 9]
對應原本數組的位址表: 
[2, 5, 3, 0, 7, 1, 6, 4, 8]
利用位址表來產生排序: 
[-2, -2, -1, -1, -1, 4, 4, 5, 8]
```
